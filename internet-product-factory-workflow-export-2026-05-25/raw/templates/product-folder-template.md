# Product Folder Template

Use this structure for every product:

```text
products/product-XXX/
  00_context/
  01_demand_discovery/
  02_demand_scoring/
  03_q_gate/
  04_product_discovery/
  05_mvp_specification/
  06_build_execution/
  07_multi_agent_audit/
  08_packaging_launch/
  09_post_launch_feedback/
  10_reflection_memory/
```

Each stage must include `stage-report.md`, `checkpoint.md`, `prompt-log.md`, and `decision-log.md`.
