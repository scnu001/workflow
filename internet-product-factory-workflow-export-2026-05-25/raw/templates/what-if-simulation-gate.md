# What-if / Constraint Simulation Gate

## Purpose

Decision-support products should show what changes when assumptions change.

## Example What-if Modes

- demand +20% / -20%
- lead time +20% / -20%
- budget limit
- price increase
- supplier delay
- lower conversion rate
- higher failure rate
- seasonality adjustment

## Rules

- If the product claims to support decisions, it should test assumption changes.
- If ranking changes easily under small assumption changes, confidence must be downgraded.
- Sensitivity results must affect recommendation confidence.
